Companies Endpoints

GET /companies
Get all companies with pagination
Query Parameters: page (default: 1), limit (default: 6)
GET /companies/type/:type
Get companies filtered by business type
Path Parameters: type (cafenea, patiserie, restaurant, etc.)
Query Parameters: page (default: 1), limit (default: 6)
GET /companies/megamenu
Get companies organized for megamenu navigation
GET /companies/:slug
Get a specific company by slug
Path Parameters: slug
GET /companies/city/:city
Get companies filtered by city
Path Parameters: city
Query Parameters: page (default: 1), limit (default: 6)
GET /companies/county/:county
Get companies filtered by county
Path Parameters: county
Query Parameters: page (default: 1), limit (default: 6)
